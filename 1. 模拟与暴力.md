# 模拟 (implementation)
## 例题
### 1. [约瑟夫环](https://leetcode-cn.com/problems/yuan-quan-zhong-zui-hou-sheng-xia-de-shu-zi-lcof/)

> $0, 1,..., n-1$ 这 $n$ 个数字排成一个圆圈，从数字 $0$ 开始，每次从这个圆圈里删除第 $m$ 个数字。求出这个圆圈里剩下的最后一个数字。
> 
> 例如, $0, 1, 2, 3, 4$ 这 $5$ 个数字组成一个圆圈, 从数字 $0$ 开始每次删除第 $3$ 个数字, 则删除的前4个数字依次是 $2, 0, 4, 1$, 因此最后剩下的数字是 $3$.

若用数组实现, 可利用 $(p + m - 1) \% n$ 实现.
```c++
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n, m;
    cin >> n >> m;

    // 该方法通常会超时, 仅作模拟的例子
    vector<int> nums(n);
    for (int i = 0; i < n; i++)
        nums[i] = i;

    int cur = 0;
    while (nums.size() > 1)
    {
        cur = (cur + m - 1) % nums.size();
        cout << nums[cur] << " ";
        nums.erase(nums.begin() + cur);
    }
    cout << endl << nums[0];

    return 0;
}
```

# 暴力 (brute force)

## 例题

### 1. 百钱买百鸡问题

> 某人有 $100$ 元, 打算购买 $100$ 只鸡. 公鸡每只 $3$ 元, 母鸡每只 $5$ 元, 小鸡三只 $1$ 元.
> 
> 求用 $100$ 元买 $100$ 只鸡, 各为多少才合适. 输出所有方案.

由题可得到如下矩阵
$$
\begin{pmatrix}
    3 & 5 & \frac{1}{3} &100 \\
    1 & 1 & 1 & 100
\end{pmatrix}
$$
得其解为 $k(-\frac{7}{3}, \frac{4}{3}, 1)^{\rm{T}} + (200, -100, 0)^{\rm{T}}\ (k \in \rm{R})$, 再在其中找符合题意的自然数解即可.
```c++
#include <bits/stdc++.h>
using namespace std;

int main()
{
    for (int z = 0; z <= 300; z += 3)
    {
        int x = -7 * z / 3 + 200;
        int y = 4 * z / 3 - 100;
        if (x >= 0 && y >= 0)
            cout << x << " " << y << " " << z << endl;
    }

    return 0;
}
```

### 4. 白学串

> 给定一个 $n$ 个数的数字序列, 每个数不超过 $1 \times 10^9$, 有 $q$ 次询问, 每次询问一个区间是否存在三个数可以组成一个三角形, 输出 `YES` 或 `NO`. ( $1\leq n, q \leq 1 \times 10^5$ )

不难发现, 一组数不能构成三角形的充要条件是该组数排序后任意三个相邻的数不能构成三角形. 不能构成三角形的极限情况是 $Fibonacci$ 数列
$$1, 1, 2, 3, 5, 8, 13, 21...$$
$Fibonacci$ 数列增长很快, 在第 $44$ 项便超过 $1 \times 10^9$, 而题目要求每个数不超过 $1 \times 10^9$, 故当 $n \geq 44$ 时, 输出 `YES`, 否则排序后判断任意三个相邻的数是否构成三角形.
